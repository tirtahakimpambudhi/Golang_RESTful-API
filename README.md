## Getting Started

### Prerequisites

- Install External Package 
    - MySQL Driver
    - Validator
    - httprouter
    - Go Dotenv

### Installation

1. Run in CMD or Bash 
    ```bash 
    go mod tidy
    ```
2. If you want to change the configuration, it's in the .env file
3. Run in CMD or Bash 
    ```bash 
    go run main.go
    ```
